# Gray Image Example

Just a simple example of how to remove colors from an image.
Excelent example to test OpenCV.

## Requirements

- Windows 10
- OpenCV 4.3.0
- PoweShell 7
- Visual Studio 2019 or Community 2019
- CMake 3.17.3

## How to run:

1. Execute compile.ps1
2. Execute run.ps1 <image_filename.jpg> or full path

Checkout the output.
